### 描述
一个 Git Clone 的插件，适用于 FlowCi

### 输入参数
>- PLUGIN_ARTIFACT_PATH `产物的地址 - 如果Path是个目录将打包上传，不是目录则正常上传`

### 输出环境变量

>如果在下个step使用输出的环境变量 请在ENV中加上 `FLOW_ENV_OUTPUT_PREFIX: PLUGIN_`， 这样的话就会把PLUGIN_前缀的环境变量传递到下个step
